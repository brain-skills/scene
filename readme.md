# Scene — Website Template

**Scene** is a modern website template developed for the upcoming **Scene Café / Theater** in **Tbilisi, Georgia**.  
The venue combines a café atmosphere with a theatrical stage, creating a cultural space for live music, performances, and artistic events.

The template provides a clean, elegant, and responsive foundation for presenting the venue online.

---

## About Scene

**Scene** is a concept venue founded by Georgian singer **Nino Basharuli**.  
The space is designed as a hybrid of a café and a small theater stage where guests can enjoy:

- Live music performances
- Theater shows
- Cultural events
- Artistic gatherings
- Café and lounge atmosphere

The website serves as the main digital platform for presenting the venue, its events, and its artistic identity.

---

## Project Purpose

This repository contains the **HTML template** for the Scene website.  
It is designed to provide:

- A modern and elegant design
- A responsive layout for all devices
- A flexible structure for future expansion
- A clear visual identity for the venue

The template is intended to be used as the foundation for the official Scene website.

---

## Key Features

- Responsive layout (desktop, tablet, mobile)
- Clean and modern UI design
- Event-focused page structure
- Menu and venue presentation
- Photo and media sections
- Contact and location information
- Performance and event announcements

---

## Planned Pages

The template is designed to support the following pages:

- **Home** — introduction to the venue
- **About** — the concept of Scene and its story
- **Events** — upcoming shows and performances
- **Menu** — café offerings
- **Gallery** — photos and media from the venue
- **Contact** — location, reservations, and inquiries

---

## Technologies

The template is built using standard web technologies:

- **HTML5**
- **CSS3**
- **JavaScript**

The structure is intentionally simple to allow integration with different back-end systems or CMS platforms.
